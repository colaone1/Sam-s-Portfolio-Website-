Event Management App

A full-stack event management application with Next.js frontend and Express/MongoDB backend. Features user authentication, event creation, and real-time updates.

Project Overview

The Event Management App enables users to register, authenticate, create, and manage events in real time. Built as a team project, it demonstrates full-stack development, RESTful API design, and modern UI/UX.

Next.js Express MongoDB JWT Real-time

Key Features

  • User registration and authentication
  • Create, view, edit, and delete events
  • Real-time event management and updates
  • Modern, accessible UI design
  • Team collaboration features
