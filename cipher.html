Cipher Encoder/Decoder

A secure message encryption and decryption tool implementing various cipher algorithms

Project Overview

This encryption tool provides a secure way to encode and decode messages using various classical and modern cipher algorithms. It features an intuitive interface and supports multiple encryption methods.

Key Features

  • Multiple cipher algorithm implementations
  • Real-time encryption/decryption
  • Key generation and management
  • File encryption support
  • Command-line interface

Technologies Used

  • Python 3.9
  • Cryptography library
  • Tkinter for GUI
  • Argparse for CLI
  • Unit testing with pytest

# Example of the Caesar Cipher implementation
class CaesarCipher:
    def __init__(self, shift_key):
        self.shift_key = shift_key % 26
        
    def encrypt(self, text):
        result = ""
        for char in text:
            if char.isalpha():
                ascii_offset = ord('A') if char.isupper() else ord('a')
                shifted = (ord(char) - ascii_offset + self.shift_key) % 26
                result += chr(shifted + ascii_offset)
            else:
                result += char
        return result
        
    def decrypt(self, text):
        return CaesarCipher(-self.shift_key).encrypt(text)
