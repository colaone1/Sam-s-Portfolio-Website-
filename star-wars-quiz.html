Star Wars Trivia Quiz

A fun and interactive Star Wars trivia quiz built with HTML, CSS, and JavaScript. Test your knowledge of the Star Wars universe with this engaging quiz application.

Project Overview

This interactive quiz application challenges users with Star Wars trivia while implementing modern web development practices. The project showcases responsive design, interactive elements, and efficient state management.

Key Features

  • 5 challenging Star Wars trivia questions
  • 60-second timer per question
  • Randomized question order
  • Visual feedback with themed images
  • Progress tracking
  • Responsive design
  • Star Wars themed UI with yellow accents
  • Interactive elements with hover effects

Technologies Used

  • Vanilla JavaScript for functionality
  • Tailwind CSS for styling
  • HTML5 for structure
  • Responsive design principles
  • Modern CSS animations

Question Categories

Lightsaber Colors
